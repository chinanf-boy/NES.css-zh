## 安装

### CDN

```html
<!-- minify -->
<link
  href="https://bcrikko.github.io/NES.css/css/nes.min.css"
  rel="stylesheet"
/>
```

OR

```html
<!-- non-minified -->
<link href="https://bcrikko.github.io/NES.css/css/nes.css" rel="stylesheet" />
```
